### TALA-alternative layout R&D (2026-06-18 — build a compact-packing engine we control)
Cluster from the d2/TALA dig: TALA's look (compact 2D packing **+** clean edge routing) is unreachable via d2's bundled engines (dagre / elk-via-5-flags) or mermaid-dagre. Five options, cheapest→heaviest; **do A first**, escalate only if it falls short. **B, C, E are the [Adaptagrams](https://github.com/mjwybrow/adaptagrams) family in JS** — cola.js (B) = MIT reimpl of libcola+libvpsc, libavoid-js (C) = LGPL WASM port of libavoid, HOLA (E) = libdialect (unported); Adaptagrams' own bindings are Java/Python only. All spike-first, lazy-loaded (zero-cost unless used), benched against TALA on the shared complex graph. **Each carries a license-analysis gate** (can we ship it in a distributed extension?) — permissiveness drops A→D: elkjs **EPL-2.0** (weak), cola **MIT** (clean), libavoid **LGPL** (✅ shippable via §6b — keep wasm separate/no-inline), OGDF **GPL** (likely blocker). Clear the licence before vendoring.
